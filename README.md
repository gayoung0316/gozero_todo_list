# 수박 두잇

혹시 다들 수박 게임을 들어보신 적 있으신가요? 

수박 게임처럼 크기가 다양한 동그라미들로 오늘 해야할 일을 가득 채워보세요! <br>
크기가 클수록 중요한 일이고, 작을 수록 여유를 가지고 해도 된답니다. <br>
원하는 색깔로 알록달록 자유롭게 다채로운 동그라미를 만들 수 있어요!

일을 다 마치셨다구요?
다 마친 일을 탭하여, 터트려 보세요!
오늘 가장 먼저 끝낸 세 가지의 일은 명예의 전당에 자랑스럽게 들어간답니다!
